## Quick Start

Download the zip file or Clone this repo, using:

```bash
git clone https://github.com/MadAitcH/jbook_local_client
```

Install the dependencies:

```bash
npm install
```


## Run App

```bash
# to run server
npm start

# to build the client
npm run build
```
